**Description**

The code provided in this folder are related to the project "_A novel L-shaped refinement chain cuts method for two-stage stochastic programs_" (see reference below).

Specifically, we provide the following files:
  - Distance_matrix_50bins.csv contains the distances between each location
  - Filling_level.csv contains, for each bin and for each collection day, the filling level recorded by the waste collector of _ERSUC - Resíduos Sólidos do Centro, S.A._

**Reference**

M. Hewitt, F. Maggioni, and A. Spinelli (2026). _A novel L-shaped refinement chain cuts method for two-stage stochastic programs_. Under review. Preprint available at https://arxiv.org/abs/2606.02469
